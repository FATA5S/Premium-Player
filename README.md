Плеер для windows. Поддерживает mp3, wav. ogg и flac формат песен.
Чтобы загрузить песни в плеер, вам надо нажать на кнопку "папка"→ выбрать папку с песнями→ плеер считает содержимое папки и добавит песни в плейлист справа. Если в этой папке будет что-то кроме тех форматов которые я указал, то плеер просто их не увидит. После каждого открытия плеера песни надо будет добавлять по новой.


Так же плеер имеет визуализацию частот от 30 до 190гц и подстраивается под цвет обожки. Если обложки нет, то будет еле заметный лунный цвет.
